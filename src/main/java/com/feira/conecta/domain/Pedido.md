O que o Pedido faz de diferente
O Pedido é onde comprador e anúncio se encontram. Mas diferente das outras entidades, ele tem efeitos colaterais — quando um pedido é confirmado, o anúncio precisa ser marcado como VENDIDO. Uma ação dispara outra.
O ciclo de vida é:
Comprador faz pedido → PENDENTE
Vendedor confirma   → CONFIRMADO → anúncio vira VENDIDO
Pedido finalizado   → FINALIZADO
E as regras críticas são:
Só COMPRADOR pode fazer pedido
Não pode pedir de anúncio VENDIDO
Não pode pedir quantidade maior que a disponível no anúncio
Não pode fazer pedido do próprio anúncio
Só o dono do anúncio pode confirmar o pedido
